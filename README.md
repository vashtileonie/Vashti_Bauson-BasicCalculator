# Mini Project 2
---
A simple book library system that incorporates OOP principles. It handles two different book types: E-Books and Physical Books.


## How to Build

```bash
mvn clean install
```

## How to Run

```bash
 java -jar target/mini-project-2-1.0.1.jar 
```

## How to Test

```bash
 mvn test 
```
